Three major categories of Linux kernel versions

  • linux-X.Y -- Mainline kernel
  • linux-X.Y.Z -- Name it
  • linux-X.Y-rcN -- Release candidate for X.Y
There are other types of Linux kernel versions (e.g. -mm kernels), but to avoid overcomplication, they are not covered here.

Four major categories of Linux kernel patches

  • linux-X.Y.tar.xz -- Name it
  • linux-X.Y.Z.tar.xz -- Name it
  • linux-X.Y.Z.tar.xz -- Name it
  • linux-X.Y.Z.tar.xz -- Name it
