== SETUP & USE ==
* Use web hosting that supports PHP5, e.g. Bluehost or GoDaddy.
* Copy the contents of the SM230/ directory into your website's directory on the hosting.
* Adjust settings and create pages and menus using settings.php (the comments are inside of the file).
* Create pages as .html files in pages/ directory.
* Enjoy.
